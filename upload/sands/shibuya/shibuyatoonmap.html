渋谷toonMap

(2005年8月版）
渋谷電力館でワールドアニメエクスポが開催中！せっかく渋谷にいくならtoonなお店を回ってみよう。

電力館　web

「ワールドアニメエクスポ」開催中！（7月21〜8月31日）
8月28日にはパフィカラのライブ。

電力館のサイトにあるクイズに答えて、正解チケットをもってゆくと、記念品がもらえる。
・・・と言うことが2005年夏にやっていました。今は特にtoonと関係ありません。

ブリスター　web

アメコミ、洋トイショップ。

パワパフやTeenTitansのアメコミをはじめ、さまざまな海外物グッズがいっぱい。
パワパフの映画スタチューもあるよ。ハッピーツリーフレンズのグッズも。


ディズニーストア　web

ディズニーの公式ショップ。

店そのものがテーマパークのような作りになっていて楽しい。
当然ディズニーグッズもいっぱい！・・・・なんだけど、ミッキー系ばかりで「キム・ポッシブル」などのグッズは全然ないんだよね。
マジカルコレクション（キム）はむしろブリスターに置いてある。


タワーレコード　web

ミュージックストア。

ここの7階本屋が隠れたアメコミスポット。
アメリカンオルタナティブコミックや、サブカルよりな洋書など多数そろえる。パワパフコーナーもあるんだよ〜！！

エル・トリート　web

メキシコ料理レストラン。

直接toonやアメコミとは関係ないけど、カートゥーンに良く出てくるタコスやナチョが食べられるお店。
toon気分にひたったら、ぜひタコスでランチを。ブーヤー！

the COMIX　web

アメコミショップ。（2006年5月閉店）

ブリスターにお目当てのアメコミが売り切れていたら、こちらもぜひ寄ってみよう。品揃えがブリスターとはまた違ったチョイスなので、意外な掘り出し物も。
2階にはグッズもあり、セール品も多い。

 

ソニープラザ　web

輸入雑貨ショップ。

いろんな海外キャラのグッズなどがあったりする。
パワパフジュースがあるのもこの店。

渋谷には2店あって、渋谷109とマークシティのそれぞれ地下。
109はかなりギャル色が強くて、落ち着かないという人はマークシティにいってみよう。ちょっぴり年齢層が上のムードだ。

HMV　web

ミュージックストア。

ここも洋書なども扱ってはいるが、むしろ特筆すべきはDVDコーナー。
「海外アニメ」コーナーがあり、とても充実している。店頭じゃめったにみないあんなやつやこんなやつ、棚にそろっているのはとても楽しい。

PORD　web

キャラクターグッズショップ。

カプセルトイや食玩をはじめ、さまざまなキャラクターグッズがみっちりつまったジャンクな店。

日本のものが多いが、海外トイの掘り出し物もある。

スウォッチストア渋谷 東急東横店　web

swatchの子供向けブランドflck flak が置いてある。

パワパフ時計があるので、目の保養としてもちょっと覗いてみよう。

駅の外、ハチ公口から南口に抜ける通路の途中に、東急の出店としてあります。

 

その他

ビッグエコー　web

カラオケ屋。パフィカラ応募期間中なら投稿CDが無料でつくれる。「HiHi」や「TEEN TITANS THEME」、「Call Me, Beep Me!」など歌える。

Loft　web

キャラクター雑貨にクルテクグッズがあったりする。

 

109-2　web
デイジーラバーズやエンジェルブルーなんかのチャイルドブランドショップがあったりするんだけど、特にtoonとは関係なし。昔ここにパワパフの巨大ポスターが飾られた・・・という思い出。

 

チェコアートセレクトショップ　『ano』　web

チェコアートショップ。チェコアニメのグッズや資料の販売、上映会などおこなっている。

さらに原宿までくりだすと、キディランドや、マドレーヌショップ、タンタンショップなどもある。
