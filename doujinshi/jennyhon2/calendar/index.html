ジェニーさんカレンダー再び！

ティーンロボットの制作終了に伴い寂しいことに公式カレンダーも発売されなくなってしまいました。 これは「それなら自分たちで作ろう」と立ち上がった有志によるプロジェクトです。

ダウンロードしたファイルを印刷するだけではがきサイズのイラスト付き卓上カレンダーができます。 エレコムの「手作りカレンダーキット」に対応していて、下の写真のようなものができます。 カレンダーは一ヶ月ごとに全十二回公開します（毎月25日公開予定）。

zipファイルには「jpg画像」と「erd形式（エレコムらくちんプリント形式）ファイル」の二種類のファイルが入っています。 jpg画像はきれいな画質で印刷できますがご使用のプリンター環境によってはうまく印刷できないかもしれません。 エレコムらくちんプリントはソフトがプリンターを認識できれば簡単確実に印刷できますが、画質はjpg画像より低下します。 これはエレコムらくちんプリントの仕様のようです。 というわけでどちらのファイルも一長一短なのでいきなり本番の用紙に印刷せずまず何か別の紙でテストプリントすることを強くおすすめします。

なお、エレコムらくちんプリントはアンケートに答えるだけで無料・匿名でダウンロードできます。メールアドレスの登録作業もありません。

印刷手順

jpg形式用…右クリックメニューの「プレビュー」からの印刷手順

※Windows XPでの場合です。ご使用の各グラフィックソフトからも印刷は可能です。

  1. ファイルを右クリックし「プレビュー」をクリック。
  2. プレビュー画面下部の「印刷」をクリック。
  3. 「写真の印刷ウィザートの開始」ウィンドウが表示されるので「次へ」をクリック。
    「印刷の基本設定」をクリックし、印刷設定を以下のようにして下さい。

    4. 用紙の種類：はがき
    用紙のサイズ：はがき
    印刷の向き：横
    ※その他の設定はご使用のプリンターに合わせて適宜設定して下さい。

  4. 印刷設定ができたら「次へ」をクリック。
    「レイアウトの選択」は「フルページ写真プリント」を選択します。
  5. 「次へ」をクリックすると印刷が開始されます。

erd形式用…エレコムらくちんプリントからの印刷手順

  1. エレコムらくちんプリントをダウンロードしセットアップします。
  2. 初回起動時には「最新用紙情報の確認」ウィンドウが開きますが無視してかまいません。
    「エレコムらくちんプリントを起動」をクリックします。
  3. メニューの「開く」をクリックしダウンロードしたerdファイルを選択します。
  4. 「印刷」アイコンを押し印刷します。
